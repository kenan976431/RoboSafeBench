# Methods

## Baseline Methods

We implement the following baseline methods from RoboTwin:

### 1. ACT (Action Chunking Transformer)
- Efficient action prediction
- Temporal consistency

### 2. Diffusion Policy
- Multimodal action distribution
- Robust to noise

### 3. RDT (Robot Diffusion Transformer)
- Scalable architecture
- Strong performance

## Advanced Safety Methods

### SafeVLA
- Safety-aware visual-language-action model
- Constraint satisfaction

### Constrained RL
- Safety constraints during training
- Guaranteed safety properties
